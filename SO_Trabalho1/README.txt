-- Sistemas Operacionais - Trabalho 1
	O arquivo '[Elcius_Ferreira_11400968].py' referente ao projeto proposto possui as implementações dos algoritmos de ordenação FCFS (First Come First Served), SJF (Smaller Job First) e RR (Round Robin). Dadas as entradas contidas no arquivo 'entradas.txt', o programa irá analisá-las e exibirá os tempos de retorno, resposta e espera para cada algoritmo de escalonamento.
	
	Para inserção de entradas basta colocá-las no arquivo 'entradas.txt'.
