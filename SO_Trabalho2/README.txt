-- Sistemas Operacionais - Trabalho 2

	O arquivo '[Elcius_Ferreira_11400968].py' referente ao projeto proposto possui as implementações dos algoritmos de substituição de páginas FIFO (First In First Out), OTM (Algoritmo Ótimo) e LRU (Least Recently Used). Dadas as entradas contidas no arquivo 'entradas.txt', o programa irá analisá-las e exibirá a quantidade de faltas de páginas referente a cada um desses algoritmos.
	
	Para inserção de entradas basta colocá-las no arquivo 'entradas.txt'.
